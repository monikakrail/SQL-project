**📊 Tuition Centre Management System (SQL Project)**

**📌 Project Overview**

This project focuses on developing a Tuition Centre Management System using SQL. The system is designed to manage student records, track attendance, and maintain organised data efficiently using relational database concepts.

**🎯 Objectives**

Maintain and update student information

Track and manage student attendance

Organise data efficiently using SQL tables and relationships

Automate operations using stored procedures

**🛠 Tools & Technologies**

SQL (MySQL / SQL Server)

Relational Database Management System (RDBMS)

**🔍 Key Features & Functionality**

1. Database Creation & Management

Created and managed multiple tables for student data and attendance

Performed operations such as:

Create tables

Modify table structure

Delete tables

2. Data Manipulation

Inserted records into tables

Updated existing data

Altered table columns as required

3. Data Cleaning

Cleaned and structured raw data using SQL commands

Ensured consistency and accuracy of records

4. Stored Procedures

Developed stored procedures to:

Add new student records

Update student details

Manage attendance records

Improved efficiency and reduced manual effort

5. Data Relationships & Joins

Used INNER JOIN to combine data from multiple tables

Retrieved meaningful insights by linking student and attendance data

**📈 Key Outcomes**

Built a structured and efficient database system

Improved data organisation and accessibility

Demonstrated strong understanding of SQL concepts and database operations

**📂 Project Files**

SQL scripts (table creation, queries, stored procedures)

**💡 Learning Outcome**

This project helped me strengthen my understanding of:

SQL queries and database design

Stored procedures and automation

Data cleaning and data integrity

Working with relational databases in a structured way
